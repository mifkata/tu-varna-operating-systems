# Задачи по Операционни Системи

Реализация на задачи за семестриален контрол по предмет Операционни Системи в ТУ-Варна от Андриян Иванов, специалност КСТ-задочно, курс 3, група 2.

## Базово упътване

```
Задачи за реализация, съгласно ръководството за лаб. упражнения:
1. Задача от презентация в упражнение – 5т.
2. Задачи 2 и 3 от тема 5 – 5т.
3. Задача 3 от тема 7 – 5т.
4. Задача 2 от тема 8 – 5т.
5. Задача 2 от тема 9 – 10т.
6. Задача 3 от тема 9 – 10т.
7. Задача 3 от тема 12 – 10т.
```

Задачите са реализирани в съответните директории `app1` за първа, `app2` за втора и тн.

## Make команди

```sh
make devcontainer   # стартира dev контейнера и отваря zsh терминал
make build <N>      # компилация на задача N (напр. make build 1)
make run <N>        # стартиране + компилация на задача N (напр. make run 1)
```
