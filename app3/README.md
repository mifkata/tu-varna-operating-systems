## Задание

Реализирайте програмно проблема "Прозиводител - Консуматор". Производителят генерира азбуката от "a" до "z" и я записва в общ буфер с размерност **char**. Консуматора чете от буфера и извеждана на екрана поредната буква от буфера.

## Make команди

```sh
make            # компилация на програмата
make run        # стартиране + компилация, ако не е намерено binary
```
