# Задача 2

```
2. Задачи 2 и 3 от тема 5 – 5т.
```

## Задание

1. Напишете С програма за сортиране на въвеждани от клавиатурата 10 цели числа. Създайте програмата в един файл. Програмата да съдържа 3 функции: за въвеждане на данните, за сортирането им и за тяхното извеждане.​

2. Модифицирайте предната програма като дефинирате всяка от функциите в отделен файл.​

3. Създайте Makefile за да компилирате файловете от предходната задача.

## Make команди

```sh
make            # компилация на програмата
make run        # стартиране + компилация, ако не е намерено binary
```
