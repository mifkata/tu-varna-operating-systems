# Dev Container

Среда за разработка, базирана на Ubuntu 24.04 (linux/amd64), конфигурирана за работа с C програми.

## Какво съдържа

- GCC, G++, Make, CMake, GDB, strace, ltrace
- Git, Vim, Nano, Zsh с Oh My Zsh
- VS Code разширения за C/C++ и Makefile

## Команди

Всички команди се изпълняват от директорията `.devcontainer/`.

```sh
make            # стартира контейнера и отваря zsh терминал
make start      # стартира контейнера във фонов режим
make stop       # спира контейнера
make restart    # рестартира контейнера
make build      # билдва Docker образа
make rebuild    # билдва Docker образа без кеш
```

## Работа с проекта

След влизане в контейнера, работната директория е `/workspace`. Оттам може да се използват следните команди:

```sh
make <N>        # компилация на задача N (напр. make 1)
make run <N>    # стартиране на задача N (компилира ако няма binary)
```

Задачите се намират в директориите `app1/`, `app2/`, и т.н. Всяка директория си има собствен `Makefile`:

```sh
make            # компилация
make run        # стартиране
make clean      # изтриване на компилирани файлове
```
