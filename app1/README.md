## Задание (Lab 7)

1. Да се напише програма на С, която изпълнява два вложени цикъла for с итерации до 1 000 000. ​

2. Да се стартира програмата в background​

3. Да се стартира програмата нормално в командния ред. Да се определи нейния PID и да се унищожи процеса по показаните начини

## Make команди

```sh
make            # компилация на програмата
make run        # стартиране в бекграунд + автоматично убиване след 3 секунди
```
